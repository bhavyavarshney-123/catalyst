# Catalyst

AI-powered career intelligence platform built in Go.

Catalyst ingests recruiter communications and interview updates,
creating a searchable semantic memory layer and generating
actionable career insights.

## Tech Stack

- Go
- PostgreSQL
- pgvector
- OpenAI
- LangGraph
- Docker
- AWS

## Roadmap

- Gmail Integration
- Metadata Extraction
- Semantic Search
- RAG Pipeline
- LangGraph Workflows
- Career Intelligence Reports
